## How to run

```bash
# install dependencies
$ npm install
# Tailwind CSS build
$ npm run watch:css
# serve with hot reload (Auto-refresh) at localhost:3000
$ gulp build
```

**Attention**: ```npm run watch:css``` & ```gulp build``` must run parallel
